Secure your website with MYDIGIPASS.COM

VASCO promotes the contribution of source code and regularly adds feature code to its master branch. This makes it easier for developers to integrate our software.

To start, create an account on http://developer.mydigipass.com. From there, you can use your own sandbox environment to test your integration. The following plugins are freely available:

  • ruby
  • wordpress
  • magento
  • drupal (awaiting approval) [demo]

OAuth2 API Guide

To secure the user accounts of your website with MYDIGIPASS.COM, you need to integrate our OAuth2 secure login API. The documentation can be downloaded here. Check our developer site regularly for the latest version.
