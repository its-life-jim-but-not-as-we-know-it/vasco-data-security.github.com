Secure your website with MYDIGIPASS.COM

VASCO Data Security offers open source software to make integrating with MYDIGIPASS.COM easier.

First create an account http://developer.mydigipass.com, and there you can create a sandbox environment to test your integration. Then use one of our supported plugins for

  • ruby
  • wordpress
  • magento
  • drupal (awaiting approval) [demo]

Forking

If you have any improvements, you can always fork and send pull requests.

If you need any support, checkout our support-site, or open a github issue.
